# UM - Core/Starter Theme

UM Core/Starter theme is a multipurpose theme. Provide/suggest basic skeleton for Responsive themes, and Framework to its childs.
Features include custom headers, theme options, and more customization available via UM-Plug Plugins.

## UM-GUI

UM-GUI is collections of javascripts(jQuery) and CSS3 to make this theme standouts and easy-to-develope.
Functions is inherited from the UM-PLUG Plugins(https://github.com/tacoen/um-plug).
